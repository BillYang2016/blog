闪灵童话铁兽与闪灵童话

在前篇的闪灵童话铁兽思路中，通过三轴互相调度，做出终场为淘气精灵+ip+同盟车+蜂鸟（破坏剑）+抗战的终场
这个终场足够立体，但是存在以下问题：

  1. ip若被无效，场上格子数不足以开出抗战
  2. ip+破坏剑的1800攻并不是一个很可观的攻击力，容易被解失去所有阻抗
  3. 手坑不够多
  4. 容易被三轴自肃互卡而产生问题最终抛弃其中一轴的终场
  5. 为了场面立体而立体，浪费了大量资源，没有充分利用小动物的优势
  6. 狮鹫2500打点容易被过

本质上闪灵童话铁兽是一个展暴卡组，而原文给出的妥协场也与铁兽轴没有任何关系
那么为什么不干脆抛弃铁兽，完全发挥出小动物的优势呢？
通过童话动物群各位大佬的研究，可以发现小动物袋鼠单卡可以在对面做出很大的终场：

  1. 加速同调出L10~L12（甚至有10+次加速同调出卫星战士的表演向combo）
  2. 加速link出link5~link6（全素材连神龙或到临者）

理论上小动物可以做的终场应该非常丰富（ip狮鹫仅仅是其中的一种），但很可惜在当前环境内强大的终场不够多，实战意义较大的即为加速耀变（或加速卫星战士/鲜花）以及加速连神龙。
加速连神龙可以在对面回合掀桌，并且全抗5000打点可以下回合直接攻击，但阻抗不够丰富，会将手牌打空。至此ip加速link思路被笔者彻底搁置，link怪还是缺乏足够站场的终端（神弓在当前环境没有什么用）
加速鲜花则存在一个问题：加速是被动且慢数步连锁的，当做出鲜花时很可能已经浪费掉了许多阻抗。
非表演向加速卫星战士也能充分掀桌，但是也会耗费大量资源、额外消耗太大，卫星战士没有抗性，如果被过场很难再次续航。

因此四次加速同调出耀变应该是小动物目前最成熟、理想、上限较高的思路，combo细节将在后文给出。

卡组构筑

主要combo

河狸+1卡=加速耀变！

通招河狸一变二，叠放巨大喷流，效果拉出僵尸带菌。
僵尸带菌和巨大喷流link水机，水机拉出垂耳兔。
僵尸带菌洗牌苏生，和垂耳兔叠放森之，效果检索袋鼠。
森之和水机link淘气精灵，淘气精灵任意苏生，ep跳袋鼠。

终场：淘气精灵+袋鼠+1怪

非河狸初动也可以通过闪灵初动完成，拉出红色闪灵下级护航，最终淘气精灵可以苏生一个红色闪灵做康。
若手牌资源足够，可以将森之留场。

扩大化终场：淘气精灵+袋鼠+森之+红色闪灵下级

对手回合：（耀变+1抽2弹）

  1. 袋鼠变小猫与小马
  2. 小马回收垂耳兔、小猫检索小海豹（排出连锁如果耀变失败也有充足阻抗）
  3. 淘气苏生水机，水机加速小车抽1
  4. 垂耳兔与场上的二星非调整怪加速出欢快、连锁海豹效果加速出兽神（优先丢掉袋鼠）
  5. 欢快c1弹一怪、兽神c2弹一怪（自己的对象选欢快），小车c3加速
  6. 小车将兽神与欢快三角加速同调出耀变

若初始手牌5张，做出耀变后手牌仍为五张，同调额外仅消耗4张，耀变每回合续航
可以更换为1弹2抽（使用星尘抽1战士）

妥协场

适用于吃G或做最大场被停

通招河狸一变二，叠放森之找袋鼠，ep跳袋鼠。
终场：森之+袋鼠

对手回合：

  1. 袋鼠变小狗小猫
  2. 小狗拉月耳兔炸卡，小猫拿垂耳兔
  3. 垂耳兔与月耳兔加速出欢快一弹（或虹光一康）
  4. 小动物回手时森之取对象一擦（可以通过三次分别回手合理利用该效果）

仅消耗1张手牌做出三个阻抗，若吃G对手抽3（其中1抽不可避免）
是一个相当优质的妥协场

小动物无限资源续航

不要跟小动物残局拖续航，因为这部分将阐述小动物如何不消耗资源做到每回合两阻抗
条件：袋鼠已调度出小猫小狗

  1. ep跳小猫小狗
  2. 小狗拉月耳兔炸卡，小猫拿垂耳兔加速同调
  3. 欢快根据自己的效果回到额外，小狗小猫已回手
  4. 若存在捉迷藏（森之可以检索），每回合回收墓地三张兽族抽1
  5. 若存在僵尸带菌和小马，可以回收月耳兔并使用僵尸带菌洗回
  6. 自己回合童话动物叠放兴奋，对手回合解体特招墓地小动物/月耳兔炸卡

实际资源消耗量：0
每回合一炸一弹或一康，十分恐怖兄弟

后攻突破

若对手复数康，最好的突破仍然是不入连锁神骑书呆高达
兴奋也可以拔除素材直击，但兴奋需要两只兽族怪兽较为困难

值得注意的是，在高达突破后，小动物可以自己ep跳出，对面回合回手避免被自己高达送墓，不需要做场即可达成自然阻抗

总体上而言卡组后攻突破能力偏弱，因此很依赖系统外的手坑和解场功能卡

单卡分析

大宇宙人

不知读者是否注意到，小动物轴与大宇宙人完美适配。

大宇宙来咯，这这大宇宙都来了怎么还不吃呢
次元吸引者，有人说小动物都回家了，不会去除外区
tnnd，跟我玩阴滴是吧

大宇宙对当前主流环境几乎是毁灭性打击，打击范围覆盖青蛙闪灵、双子闪灵、珠泪、黄金国
打击范围不包括救祓和旅鸟，但是救祓和旅鸟会被闪灵体系压制
因此携带大宇宙人理论上可以几乎与任意主流扳扳手腕，那么此卡就没有不入主卡的理由了，主卡下2

听我说谢谢你，大宇宙人，因为有你，温暖了四季

最新的地属性dp盒子给出的守墓新魔法卡同样值得关注，该卡将是先攻替代大宇宙人的优质单卡，但仍需要观察测试。

闪灵本家轴

3蓝喷3白喷1怪康1魔陷康3到来1粉碎
额外2淘气1巨大

闪灵最大轴，巨大喷流只携带1因为苏生他仍然可以发效果

不久前笔者使用的是伽马暴而非粉碎，但实际对战中伽马暴起到的作用大多为快速斩杀（win faster），但由于上述无消耗续航combo的存在，若不使用伽马暴也是能够赢下的。且伽马暴对雷精内战作用很小，因此在单卡强度上便不如粉碎了。

小动物轴

1小马1小狗1小猫1小袋鼠1捉迷藏1垂耳兔1小海豹1月耳兔
额外1森之1兴奋1欢快

很理想的小动物轴
捉迷藏除了回收资源，也会给兽族一回合一次效破抗性（不吃幽鬼兔），不能轻易忽视。但捉迷藏不会优先检索，只能期待其上手（不期待也可以）。
小马是耀变必要组件，也可以从墓地回收小动物或回收河狸等
兴奋虽然出场条件较难，但可以通过欢快拉出来，也可以残局达成无限combo
月耳兔可以视为唯一纯废件，压缩其数量到1

瞪眼因为亏卡所以不再携带

初动与辅动

3河狸1僵尸带菌

河狸是小动物最优初动点（兽族），其可以苏生墓地也可以拉卡组，下3也等于0废件。但也因此同时吃到灰流丽与屋敷童，可谓又爱又恨。
僵尸带菌是水机调度出耀变组件的关键，可以被小鹿替代，下面讲一讲小鹿与僵尸带菌各自的优缺点：

  1. 小鹿是兽族，可以cost手牌拉cost；拉墓地的效果若墓地没有兽族则不能发动，地属性调整是魔救的优秀ntr目标
  2. 僵尸带菌自己可以反复横跳，可以洗回废件（如月耳兔）；不能帮助扩大兽族场面

总体而言，僵尸带菌是小鹿的上位替代。
若使用小鹿，则一般需要携带2月耳兔避免废件上手导致小狗无法炸卡的问题。

手坑

2大宇宙人3灰3G1DD乌鸦1遮2泡

共12手坑，数量中规中矩
这些手坑是笔者反复斟酌后得出的结果，部分并不优质的手坑（如效遮）主卡下1喂给剑指
这里回答几个问题：

  1. 为什么不携带幽鬼兔？因为幽鬼兔必须要送去墓地发动而非丢弃，在卡组中存在大宇宙人的情况下，幽鬼兔显得并不好用。另外若幽鬼兔可以拦停闪灵内战，则任意一个其他手坑也可以拦停，只是幽鬼兔会显得更赚，这部分差额可以用小动物自然阻抗弥补。
  2. 泡影携带2张？泡影不携带三张是因为卡位不够，主卡卡位非常紧张几乎没有位置投入第三张泡影了。泡影应该是当前环境数一数二的手坑，因为闪灵护航一般会优先选择怪康护航从而遗漏泡影。
  3. DD乌鸦？乌鸦可能是未来将代替幽鬼兔的存在，其泛用性可以覆盖珠泪，而幽鬼兔只会帮助珠泪。备牌会投入更多DD乌鸦以供换备需要。
  4. 浮幽樱问题？小动物闪灵可能是最不会被浮幽樱打停的卡组了，因为小动物基础combo不需要任何闪灵组件，备牌携带两张浮幽樱以针对其他闪灵。

功能卡

1三战之才3指头

三战是如今手坑横行的时代很厉害的功能卡，几乎上手就可以触发，就算不使用也有下1喂指的必要

加速同调组件

额外1耀变1psy1兽神1欢快1虹光1小车1水机

psy可以通过2+6做出（小车+兽神/垂耳兔+兽神），可削手可回收资源，但出场率较低可以换出
虹光是对战珠泪利器，也是以前未强化时的小动物绑定卡，但身材较小出场率也较低可以换出

高达组件

额外1高达1书呆1神骑

神骑被即将发售的新卡蚊子替代的可能性考察：

  1. 神骑伤判弹卡可以解场（优于蚊子）
  2. 蚊子在双方任意怪兽攻宣都可以发动且没有一回合一次（劣于蚊子）
  3. 蚊子更容易斩杀（劣于蚊子）
  4. 蚊子可以两只以上做素材（劣于蚊子）

如果仅仅当高达驾驶员的代替品，蚊子一定优于神骑。如果用于伤判解场，神骑优于蚊子。
很难抉择，需要新卡放出后测试才能得出结果。

换备思路


任何的闪灵构筑，一定要考虑到结界波问题。在当今环境中常用的反制结界波手段有：

  1. 魔封的芳香
  2. 魔法抵消器
  3. 禁止令
  4. 红莲之指名者
  5. 神字反击

魔法抵消器是魔救常用思路，很可惜对闪灵没有参考意义
在剩下的4种手段中，最有效的应该是神字反击了，一般选择神宣
神宣不仅可以反制结界波，还可以康掉闪灵的通招，使其失去跳板，泛用性很广泛。
但红莲与禁止令也是可以考虑的备牌。

可以考虑换出1张G、1张捉迷藏

卡组缺陷与结语

在与铁兽告别之后，卡组明显顺手了不少，不过仍然存在一些问题
首先是所有闪灵的通用问题，当对手投入更多的解场牌而非手坑的时候，闪灵就会非常难受，只能依靠系统外的卡片来反制。
然后则是小动物闪灵的独家问题，小动物闪灵会被环境中大量存在的次元障壁所有效牵制（对手回合宣言超量自己回合宣言同调），而次元障壁对其他闪灵则威胁不大。小动物闪灵可以通过信息差、神宣、红重、魔陷康闪灵下级来反制次元障壁，需要读者特别注意。

本文到这里就接近尾声了，从闪灵发售前的预构筑到如今的成熟构筑向探讨笔者也已经反复修正了一个多月，笔者仍然十分看好这套卡组的未来潜力，作为小动物老玩家这次好好地抱了一波闪灵的大腿，但也给闪灵带来了新的思路。小动物闪灵或许将摆脱闪灵固有的枯燥乏味的印象，变得又魂又有趣，在青蛙轴可能被刀的未来很可能大放异彩。
